# WordPress Reviews Plugin

Плагин для управления отзывами с галереей фотографий и фильтрацией.

## Установка

1. Скопируйте папку `wordpress-reviews-plugin` в директорию `wp-content/plugins/` вашего WordPress сайта
2. Активируйте плагин в админ-панели WordPress (Плагины → Установленные плагины)
3. После активации в меню появится раздел "Отзывы"

## Использование

### Добавление отзыва

1. Перейдите в **Отзывы → Добавить новый**
2. Заполните:
   - **Заголовок** - название отзыва
   - **Содержание** - текст отзыва
   - **Изображение записи** - основное изображение (будет использовано в галерее, если не указана галерея)
3. В метабоксе "Детали отзыва":
   - **Имя автора отзыва** - имя автора
   - **Город** - город автора (можно также выбрать из таксономии справа)
   - **Есть видео?** - отметьте, если у отзыва есть видео
4. В метабоксе "Галерея фотографий":
   - Нажмите "Добавить фотографии"
   - Выберите несколько изображений
   - Первая фотография будет использована как основное изображение в галерее на странице списка
5. В метабоксе "Видео отзыва":
   - Вставьте URL видео (YouTube, Vimeo и т.д.)
6. Таксономии (справа):
   - **Города** - выберите или создайте город
   - **Изделия** - выберите или создайте изделие
   - **Годы** - выберите или создайте год
7. Нажмите "Опубликовать"

### Просмотр отзывов

- **Список отзывов**: `/reviews/` (или настройте постоянные ссылки)
- **Детальная страница**: автоматически создается для каждого отзыва

### Фильтрация

На странице списка отзывов доступны фильтры:
- **Город** - фильтр по городу
- **Изделие** - фильтр по изделию
- **Год** - фильтр по году
- **Формат** - фильтр "Только с видео"

### Функционал

- Галерея фотографий с адаптивной сеткой
- Фильтрация по городу, изделию, году и наличию видео
- AJAX загрузка дополнительных отзывов (кнопка "Показать еще")
- Детальная страница с переключением фотографий
- Навигация между отзывами (Предыдущий/Следующий)
- Адаптивный дизайн для всех устройств

## Настройка постоянных ссылок

После активации плагина перейдите в **Настройки → Постоянные ссылки** и нажмите "Сохранить изменения" для обновления правил перезаписи.

## Требования

- WordPress 5.0 или выше
- PHP 7.2 или выше
